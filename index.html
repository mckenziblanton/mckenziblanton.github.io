Print(Finger)

Fingerprint Image Recognition

Objective

For this project, we decided to look at fingerprints and see what we were able to predict based on the images. Based on the data set we obtained, we attempted to predict gender, as well as the hand and finger the print belonged to.

Learn More About our Data Set

Our Data Set

The dataset contains 6,000 fingerprints, 10 each from 600 African subjects.

Fingerprints are labeled for gender, hand, and finger name. Alteration types are denoted as well.

Machine Learning Models

We used two machine learning models to predict the gender, hand, and finger belonging to the print.

The two we tried were a convolutional neural network and a sequential neural-network.

Sequential Neural-Network

The sequential neural network model contained 9,888 input nodes with one per fingerprint pixel, 3 densely-connected hidden layers with 128, 64, and 32 nodes, and an output layer of 5 nodes, one for each finger.

The models predicting hand and gender were identical with the exception that they contained 2 output nodes.

Convolutional Neural-Network

The convolutional neural network model is most commonly applied to analyzing visual imagery.

CNNs use a variation of multilayer perceptrons designed to require minimal preprocessing contained. Adapted model is called vgg16 and it has 3,612,672 input nodes with one per fingerprint pixel.

13 convolutional layers and 2 densely-connected hidden layers and an output layer of 2 nodes, one for each gender.
